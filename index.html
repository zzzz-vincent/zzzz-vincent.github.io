Projects

Needle: Deep Learning Library

C++, CUDA, Python, NumPy, Pytest

PyTorch-like deep learning library, with: automatic differentiation; gradient-based optimization of models; support for standard operators like convolutions (CNN), recurrent structure (RNN), self-attention (LSTM); and efficient linear algebra computation empowered by C++ and CUDA on both CPU and GPU devices.

RecognizablePoint: Point-cloud Visualization Library

Python, Pytorch, Typescript, React, CSS/HTML

A library for visualizing point clouds in both single images and video frames, it has the ability to make real-time adjustments to color and point weight, as well as perform object detection using bounding boxes and point upsampling using deep learning models. The project also includes a user interface for testing user experience.

Counterfactual Fairness in Voice Emotional Recognition

Python, PyTorch, NumPy, librosa

Research was conducted on various counterfactual methods to improve fairness in Voice Emotional Recognition models. These methods include data augmentation, model ensembles, and counterfactual logit pairing regulation. A baseline 3D ACRNN model was implemented for emotion recognition, and a pre-trained VQ-VAE model was used for voice gender conversion. Fairness metrics based on equal opportunity were also designed.

Bustub: Database Management Systems

C++

A relational database management system was developed, it includes a buffer pool with an LRU algorithm for fetching and writing pages, a concurrent B+ tree indexing system, and query executors such as sequential and index scans, insert/update/delete, aggregation, limit, and join to implement the isolation level for transactions. A lock manager with deadlock detection is also included to control the access of transactions.

File-Caching Proxy + Transparent Remote File Operations

C, Java RMI and Threading

A distributed remote file operation system was developed, it includes a RPC system with server process and a client stub library that enables remote UNIX file operations. The system also includes a proxy with check-on-use caching protocol, this feature robustly handles concurrent clients, enforces open-close session semantics, and uses an LRU algorithm for managing cached files.

Failure Recovery Two-phase Commit Systems

Java

A distributed transaction system was developed, it supports two-phase commit protocol that handles concurrent requests and deals with lost and delayed messages. It also includes timeout and resend operations. Write-ahead logging is implemented on persistent storage for failure recovery on both the server and client side.

Malloc: Dynamic Storage Allocator

C

A dynamic memory allocator which will consist of the malloc, free, realloc, and calloc functions. The allocator follows LIFO allocation policy with circle linked-lists data structure for free block. Finer granularity of fixed size free block lists are provided for to reduce external fragmentation. Internal fragmentation is reduced by eliminating footers.

Art in Objects: Alexa Skill for Art

Python, AWS

An interactive Alexa Skill was created that allows users to search for artworks by keywords. It has dynamic handlers for requests and responses, and uses AWS API Gateway and Lambda as the service backend to search results from public museums' databases with over 50,000 art pieces. The metadata of art tags is stored in Dynamo DB.

Web-based Santorini Board Game

Java, TypeScript, Armeria, React

A web-based board game was created with MVC pattern, using Java as the backend and TypeScript for the frontend. The game supports an AI player with Min-Max algorithm and Alpha–beta pruning for decision making.

Web-based Text Analysis Framework

Java, TypeScript, NLP Core, D3.js

A text analysis framework was created for NLP analysis, it has extensibility features, allowing the integration of data, analyzing, and visualization plugins. Data plugins parse data from various sources, analyzing plugins process data using Stanford CoreNLP and CloudMeaning API, visualization plugins render results using D3.js library. The framework is easy to use and allows users to perform NLP analysis on different text data.

COVID-19 Symptom Self Monitoring System

Java Spring, JavaScript, React, Bootstrap, MySQL

A web application was created to support self check-in and emergency warning notifications. The application uses MySQL for data storage and implements asynchronous email notifications for emergency warning signs. User authentication is handled using Spring Security and JWT. The interactive web pages are built using React, Redux, and Bootstrap.
